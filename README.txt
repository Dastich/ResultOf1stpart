Решение задачи предпологает использование 3 методов для обработки изначально заданного массива строк
1-ый метод PrintArray
Он выводит на экран поочередно все ячейки массива
2-ой метод CollectNumberOfString
Он высчитывает количество элементов массива, чья длина не превышает 3 символа.
В дальнейшем используется для создания нового массива, в которой будет записан результат
3-ий метод CreateArrayOfStringWith3OrLessChar
Он создает новый массив, который в дальнейшем будет выведен как результат выполнения программы
Он имеет 2 аргумента, исходный масиив и длина нового массива, в который будет записан результат.
Он повторно проверяет элементы исходного  массива и записывает в результирующий  массив только строки, чья длина не превышает 3 символа
В теле программы задается массив строк
Далее следует его вывод на экран.
После выполнения методов 2 и 3 на экран будет выведен результирующий массив.
